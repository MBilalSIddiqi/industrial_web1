# Ironclad Industrial Group

A static, multi-page demo marketing website for a fictional heavy-manufacturing and industrial-infrastructure company — *"Raw power. Solid form."* Brutalist industrial theme. Built with plain HTML, CSS, and vanilla JavaScript (no framework or build step).

**Live site:** https://mbilalsiddiqi.github.io/industrial_web1/

## Pages
- `index.html` — Home
- `about.html` — The Foundation
- `projects.html` — The Output
- `contact.html` — Protocol

## Tech
- HTML5, CSS3 (custom properties, thick borders, hard offset shadows, zero border-radius), vanilla JavaScript
- Palette: concrete grey, charcoal, safety-orange `#FF5722`, caution-yellow
- Google Fonts: Archivo Black + Space Mono
- Shared `styles.css` and `script.js`

## Run locally
No build step required — open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000   # visit http://localhost:8000
```
